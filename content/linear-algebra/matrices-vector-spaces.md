# 矩陣與向量空間

## 矩陣乘法：先看維度對不對

`(m × n)(n × p) = (m × p)`：**內側要相同 (n)**，外側是結果尺寸。

```
A(2×3) · B(3×2) = C(2×2)    ✓ 內側 3=3
A(2×3) · B(2×2)             ✗ 內側 3≠2，不能乘
```

算法：C 的第 i 列第 j 行 = A 第 i 列「點」B 第 j 行。

```
[1 2] [5 6]   [1·5+2·7  1·6+2·8]   [19 22]
[3 4]·[7 8] = [3·5+4·7  3·6+4·8] = [43 50]
```

注意：矩陣乘法**不可交換**，AB ≠ BA（一般而言）。

## 向量空間：在問「哪些方向組得出來」

核心三概念：
- **span（生成）**：一組向量的所有線性組合，能「填出」的空間。
- **線性獨立**：`c₁v₁+…+c_kv_k = 0` 只有「全為 0」這個解（沒人是別人的組合）。
- **basis（基底）**：又能 span 整個空間、又線性獨立的一組向量。
- **dimension（維度）**：basis 裡的向量個數。

直覺：basis 就是「最精簡、不重複、又夠用」的座標軸。

## 線性獨立的小例子

```
在 R²：
  v1=(1,0), v2=(0,1)  → 獨立（兩個不同方向）→ 是 R² 的 basis
  v1=(1,2), v2=(2,4)  → 不獨立！v2 = 2·v1，只 span 一條線
```

關鍵：**非零向量不一定獨立**；互為倍數就只 span 一條線。

## 子空間 (subspace) 的三項檢查

一個集合要當子空間，必須：
1. 含**零向量**。
2. 對**加法封閉**（兩個在裡面，相加還在）。
3. 對**純量乘法封閉**（乘任意倍數還在）。

```
{ (x, y) : y = 2x }      ✓ 過原點的直線，三項都過 → 是子空間
{ (x, y) : y = 2x + 1 }  ✗ 不過原點（沒零向量）→ 不是子空間
```

含「不等式」或「非齊次（=常數）」的集合通常不是子空間。

## Rank（秩）：真正「有效」的維度

rank = row reduction 後的 **pivot 個數** = column space 維度 = row space 維度。

它告訴你這組列/行裡有幾個真正獨立的方向。

## 常見誤解

- **矩陣乘法不可交換**（AB ≠ BA）。
- **非零向量不一定獨立**（互為倍數就不獨立）。
- 含不等式或非齊次方程的集合**通常不是子空間**。
- `rank(AB) ≤ min(rank A, rank B)`，**不是**兩者相乘。
- 行列式判斷可逆**只適用方陣**。

## 解題時的判斷

- 先檢查矩陣**尺寸能不能乘**。
- 問獨立/basis → 看線性組合是否只有零解、夠不夠 span。
- 證子空間 → 零向量、加法封閉、純量封閉，三項都要過。
- 用座標 → 先確認是「哪個 basis」下的座標。
