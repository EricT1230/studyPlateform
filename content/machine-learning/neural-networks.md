# 神經網路

## 神經網路是什麼？

它是一疊「線性變換 + 非線性轉折」堆起來的函數。
每一層先把輸入加權求和，再經過一個**非線性激活**，傳給下一層。

```
輸入 →[ 加權求和 → 激活 ]→[ 加權求和 → 激活 ]→ 輸出
        第一層               第二層
```

**權重 (weight)** 和**偏置 (bias)** 是要學的參數；激活提供「彎折」的能力。

## 為什麼一定要非線性？

如果沒有激活，每層都是線性，疊再多層**還是等於一條線性模型**：

```
線性 ∘ 線性 ∘ 線性 = 線性     ← 白疊了
```

加了非線性，網路才能學出曲線、折角這種複雜邊界。

## 怎麼學：前向 + 反向傳播

```
1. 前向傳播：輸入一路算到輸出，得到預測與損失
2. 反向傳播：用「鏈鎖律」從輸出往回算，得到損失對每個參數的梯度
3. 最佳化器：照梯度更新參數（梯度下降）
```

反向傳播只是「鏈鎖律的有效率版本」，把每層的局部梯度連乘起來。

## 常見激活函數

```
ReLU     max(0, x)    計算快、緩解梯度消失（最常用）
Sigmoid  壓到 0~1     兩端飽和時梯度極小，容易梯度消失
Softmax  輸出機率分布  多類別分類的最後一層
```

## 損失要配對輸出

```
分類 → 交叉熵 (cross-entropy)，常搭 softmax
迴歸 → 均方誤差 (MSE)
```

配錯損失，訓練會學得很差。

## 幾個一定要懂的詞

```
epoch     走完整個訓練集一次
batch     一次更新所用的一小批樣本
初始化    權重要打破對稱（不能全設成相同值）
```

## 為什麼權重不能全設 0？

若同一層權重都一樣，所有神經元收到相同梯度、學到**完全一樣的東西**，等於只有一個神經元——所以要隨機初始化打破對稱。

## 深層的兩個老問題

```
梯度消失：往回連乘的梯度越來越小 → 前面的層幾乎學不動
梯度爆炸：連乘越來越大 → 數值炸開、發散
```

對策：ReLU、殘差連結 (residual)、BatchNorm、梯度裁剪 (clipping)。

## 不同架構配不同資料

```
CNN          影像（抓局部空間特徵）
RNN          序列（有時間順序）
Transformer  序列，靠注意力 (attention)，現代主流
```

## 學習時的判斷

- 先看資料型態：影像、序列、還是表格？→ 決定架構。
- **激活與損失配對**了嗎（分類用交叉熵、迴歸用 MSE）？
- 訓練不穩 → 檢查學習率、初始化、正規化、梯度大小。
- 沒有非線性、或權重全初始化成同值，是常見的隱性錯誤。
