# 隨機變數與分布

## 隨機變數是什麼？

隨機變數就是「把隨機結果翻譯成數字」的一條規則。

```
丟兩枚硬幣，X = 正面數
  HH→2  HT→1  TH→1  TT→0
  X 的可能值：0, 1, 2
```

有了數字，就能算平均、變異、畫分布。

## 離散 vs 連續

- **離散**：值是可數的（0,1,2…），用 **PMF**（機率質量函數）給每個值一個機率。
  ```
  P(X=0)=1/4, P(X=1)=1/2, P(X=2)=1/4   （總和=1）
  ```
- **連續**：值是一段區間（身高、時間），用 **PDF**（機率密度函數）。
  - 規則：`f(x) ≥ 0` 且**整條曲線下面積 = 1**。
  - 機率 = **面積**，所以「正好等於某一點」的機率是 0，要問「落在某區間」。

## CDF：累積到目前為止的機率

```
F(x) = P(X ≤ x)
```

CDF 從 0 爬到 1，**非遞減、右連續**。離散時是階梯狀、連續時是平滑曲線。
它是離散與連續通用的描述方式。

## 期望與變異：分布的兩個摘要

```
離散  E[X] = Σ x·P(X=x)
連續  E[X] = ∫ x·f(x) dx
Var(X) = E[X²] − (E[X])²
```

期望是「重心」，變異是「散開程度」。但**摘要不等於全貌**：平均和變異一樣的兩個分布，尾巴可能差很多。

## 幾個一定要會的分布（連到情境）

```
Bernoulli(p)   一次成敗          E=p,      Var=p(1−p)
Binomial(n,p)  n 次裡成功幾次     E=np,     Var=np(1−p)
Geometric(p)   第幾次才第一次成功  E=1/p
Poisson(λ)     單位時間內稀疏事件數 E=λ,      Var=λ
Exponential(λ) 事件之間的等待時間  E=1/λ
Normal(μ,σ²)   鐘形、自然界最常見  E=μ,      Var=σ²
```

**無記憶性 (memoryless)**：Geometric 與 Exponential 特有——「已經等了多久」不影響「還要再等多久」。

## 標準化：把任何常態化成 Z

```
Z = (X − μ) / σ      →  Z ~ Normal(0, 1)
```

這樣就能查同一張標準常態表，不必每個 μ、σ 各有一張表。

## 聯合分布與獨立

兩個變數一起看時用聯合分布。**獨立**的判準：

```
聯合 = 邊際 × 邊際      （f(x,y) = f_X(x)·f_Y(y)）
```

能拆成乘積才獨立；拆不開就不是。

## 連續變數做變換：別忘了 Jacobian

若 `Y = g(X)`，換算 Y 的密度時要乘上變數變換的縮放因子（Jacobian）：

```
f_Y(y) = f_X(x) · |dx/dy|
```

忘了這項，密度幾乎一定算錯。

## 常見誤解

- **平均、變異相同 ≠ 同分布**，也**不代表獨立**。
- **共變異數為 0** 只說沒有線性關係；**獨立才保證** Cov=0，反過來一般不成立。
- 連續的 **PDF 可以 > 1**（它是密度不是機率），只有面積要等於 1。
- 變數變換忘了 **Jacobian** 是最常見的錯。

## 解題時的判斷

- 先分清**離散還是連續**，決定用 PMF 還是 PDF。
- 問的是機率、期望、變異、CDF，還是分布變換？
- 認得情境就套對應分布（成敗→Bernoulli/Binomial、稀疏計數→Poisson、等待→Exponential）。
- 常態題先**標準化成 Z** 再查表。
- 做變換補上 **Jacobian**；判獨立看能不能拆成邊際乘積。
