# 機率基礎

## 機率到底在量什麼？

機率是「某件事發生的可能性」，用 0 到 1 的數字表示：0 是不可能、1 是一定發生。
所有可能結果的集合叫**樣本空間**，其中一個（或一群）結果叫**事件**。

```
丟一顆骰子：
  樣本空間 = {1,2,3,4,5,6}
  事件「點數是偶數」 = {2,4,6}
  P(偶數) = 3/6 = 0.5
```

## 聯集、交集：別把重疊算兩次

```
P(A 或 B) = P(A) + P(B) − P(A 且 B)
```

那個「減掉交集」是重點：直接相加會把同時屬於 A、B 的部分算兩次。

```
一副牌抽一張：
  P(紅色)=26/52, P(K)=4/52, P(紅色 K)=2/52
  P(紅色 或 K) = 26/52 + 4/52 − 2/52 = 28/52
```

## 條件機率：在「已知 B 發生」的世界裡看 A

```
P(A|B) = P(A 且 B) / P(B)      （需 P(B) > 0）
```

直覺：B 已經發生，等於把樣本空間**縮小到只剩 B**，再看 A 佔多少。

```
已知骰子點數 > 3（即 {4,5,6}），問是偶數的機率：
  P(偶數 | >3) = P({4,6}) / P({4,5,6}) = (2/6)/(3/6) = 2/3
```

## 獨立：知道 B 對 A 完全沒幫助

```
A、B 獨立  ⇔  P(A 且 B) = P(A)·P(B)  ⇔  P(A|B) = P(A)
```

例：丟兩次硬幣，第一次正面不影響第二次——兩次獨立。

## 全機率：先分類，再加權

把樣本空間切成互斥又涵蓋全部的幾塊 B₁,B₂,…，就能「先拆再合」：

```
P(A) = P(A|B₁)P(B₁) + P(A|B₂)P(B₂) + …
```

例：兩台機器各生產一半的零件，良率不同 → 先按機器分類，再加權算整體良率。

## 貝氏定理：把條件方向反過來

已知 `P(B|A)`，想求 `P(A|B)`：

```
P(A|B) = P(B|A)·P(A) / P(B)
```

經典「偽陽性」直覺：就算檢驗很準，若疾病本身很罕見（base rate 低），
驗出陽性後「真的有病」的機率仍可能很低——因為健康的人多，偽陽性的絕對數量壓過真陽性。

## 期望值：長期平均會落在哪

期望值 = 每個結果乘上它的機率，加總起來，是「重複很多次的平均」。

```
E[X] = Σ x·P(X=x)

骰子：E[X] = (1+2+3+4+5+6)/6 = 3.5
```

線性性（超常用，不需要獨立也成立）：
```
E[aX + b] = a·E[X] + b
E[X + Y]  = E[X] + E[Y]
```

## 變異數：平均附近散得多開

```
Var(X) = E[(X − E[X])²] = E[X²] − (E[X])²
Var(aX + b) = a²·Var(X)      （平移不影響、縮放平方放大）
```

## 常見誤解

- **互斥 ≠ 獨立**：互斥是「不能同時發生」，獨立是「互不影響」。兩個非零機率事件若互斥，反而**一定不獨立**（一個發生另一個就不可能）。
- `P(A|B)` 與 `P(B|A)` 通常**不相等**（檢出陽性 vs 真的有病）。
- **PDF 可以大於 1**；連續變數「正好等於某一點」的機率是 0。
- **共變異數為 0** 只代表沒有線性關係，一般**推不出獨立**。
- 中央極限定理講的是「樣本平均」的分布趨近常態，**不是單一觀測值**會變常態。

## 解題時的判斷

- 先界定樣本空間與事件，畫出有沒有重疊。
- 看到「已知…」就是條件機率，先確認分母是哪個事件。
- 能切成互斥分類 → 全機率；要反轉條件方向 → 貝氏。
- 「至少一個」常用補集（1 − 都不發生）更快。
- 算期望/變異記得線性性與 `Var(aX+b)=a²Var(X)`。
