# 估計與假設檢定

## 在做什麼？

我們看不到母體的真參數（例如全台灣的平均身高 μ），只能**用樣本去猜**。
這章在做兩件事：**估計**（猜一個值或一段範圍）與**檢定**（判斷某個說法站不站得住）。

## 估計量 vs 估計值

- **估計量 (estimator)**：抽樣前的「規則」，本身是隨機變數（例如「樣本平均」這個公式）。
- **估計值 (estimate)**：把實際資料代進去後得到的**一個數字**。

```
規則：θ̂ = (X₁+…+Xₙ)/n        ← 估計量（隨機）
資料：θ̂ = (170+165+180)/3 = 171.7  ← 估計值（一個數）
```

## 好的估計量看三件事

```
不偏 (unbiased)     E[θ̂] = θ      平均而言猜中（不代表每次都準）
一致 (consistent)   n 變大 → θ̂ 收斂到 θ
有效率 (efficient)  變異越小越好
```

**標準誤 (standard error)** = 估計量抽樣分布的標準差，衡量「這個猜法有多晃」。

## 信賴區間：給範圍，不只給點

95% 信賴區間的正確意思是：**重複抽樣、每次都照同樣方法做區間，長期約 95% 的區間會蓋住真參數**。

```
95% CI:  θ̂ ± (臨界值) × 標準誤
```

不是「真參數有 95% 機率落在這個區間」——真參數是固定的，會變的是區間。

## 最大概似估計 (MLE)

問：「哪個參數值，最可能產生我手上這筆資料？」
MLE 就是讓**概似函數 (likelihood) 最大**的那個參數。

```
直覺：硬幣丟 10 次出現 7 正 → 讓「出現 7 正」機率最大的 p 是 0.7
```

## 假設檢定：先立兩個對立說法

```
H0（虛無假設）：預設、保守的說法（如「新藥沒效」）
H1（對立假設）：想證明的說法（如「新藥有效」）
```

流程：算**檢定統計量** → 看它在「H0 為真」下有多極端 → 太極端就**拒絕 H0**。

## p-value 是什麼

**p-value = 假設 H0 為真時，得到「至少像現在這麼極端」結果的機率。**
p 很小 → 資料和 H0 很不合 → 傾向拒絕 H0（通常以 α=0.05 為界）。

它**不是**「H0 為真的機率」。

## 兩種錯誤

```
            H0 其實為真      H0 其實為假
拒絕 H0      型一錯誤 (α)      ✔ 正確（power）
不拒絕 H0    ✔ 正確           型二錯誤 (β)
```

- **型一錯誤**：冤枉了對的 H0（誤報）。機率 = 顯著水準 α。
- **檢定力 (power) = 1 − β**：當 H1 真的成立時，成功抓出來的機率。

## 偏差–變異拆解

```
MSE(θ̂) = Var(θ̂) + [Bias(θ̂)]²
```

估得準不準，同時受「晃動大不大 (Var)」和「系統性偏不偏 (Bias)」影響——有時容忍一點偏差換更小的變異反而整體更準。

## 常見誤解

- **p-value 不是 H0 為真的機率**，也不是「結果由運氣造成的機率」。
- 樣本變異數除以 **n−1**（不是 n）：用樣本平均估計時損失了一個自由度，要修正。
- 樣本數變大讓區間變窄，但**消不掉模型假設本身錯**造成的偏差。
- **多重檢定**會累積偽陽性：做很多次檢定，總有幾次「碰巧顯著」，需要校正（如 Bonferroni）。
- 「不拒絕 H0」**不等於**「H0 成立」，只是證據不足。

## 解題時的判斷

- 母體變異數已知嗎？未知且近似常態 → 用 **t 分布**。
- 檢定是**單尾還是雙尾**？看 H1 的方向。
- 信賴區間與雙尾檢定**互為對偶**，可以互相交叉驗算。
- 比較估計量好壞 → 用 **MSE = Var + Bias²**。
- 寫結論時把樣本獨立性、近似條件講清楚。
