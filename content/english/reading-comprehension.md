# Reading Comprehension

## 核心概念
閱讀題要抓作者真正支持的結論，而不是把一句話過度延伸。常見考點包含 main idea、inference、tone、detail、paraphrase、cause-effect 與 argument structure。

## 解題重點
看到 `However`、`Nevertheless`、`but`，通常表示轉折。例：`The model scored well. Nevertheless, deployment was delayed.` 重點不是模型差，而是仍需更多驗證。`on the condition that` 表示「條件是」，例如 `The patch was accepted on the condition that the API remain stable.`

## 常見陷阱
不要把「可能」讀成「必然」，也不要把單一指標改善讀成整體成功。代名詞如 `it` 若不清楚，要找最明確的 antecedent。

## 練習前檢查
先用一句話寫出段落主旨，再刪掉過度絕對、與原文相反、或原文沒提到的選項。
