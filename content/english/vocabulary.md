# Technical Vocabulary

## 核心概念
工程英文常考單字在語境中的意思。不要只背中文翻譯，要能放進句子判斷語氣與行動含義。

## 解題重點
`Deprecated` means supported but discouraged: `This API is deprecated; use v2 instead.` 表示仍可能可用，但不建議繼續依賴。`Idempotent` means same result after repeated runs: `Retrying this migration is safe because it is idempotent.` `Preliminary` 是初步，`conclusive` 才是定論。

## 常見陷阱
`Bikeshed` 不是腳踏車棚本身，而是花太多時間爭小細節。`Accountable` 偏向負責任、可追蹤，不是單純「會計」。`Tradeoff` 表示取捨，不等於缺點。

## 練習前檢查
把單字代回句子，看是否符合前後文的轉折、限制與行動建議。
