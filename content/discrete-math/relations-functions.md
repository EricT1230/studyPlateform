# 關係與函數

## 關係是什麼？

關係就是「一堆有序配對 (a, b)」，描述元素之間「有沒有某種連結」。
像「a 整除 b」、「a 和 b 同餘」、「a ⊆ b」都是關係。

要檢查的四個性質（複習）：反身、對稱、反對稱、遞移。兩個重要組合：
- **等價關係** = 反身 + 對稱 + 遞移
- **偏序** = 反身 + 反對稱 + 遞移

## 等價關係 → 把集合「切成不重疊的群」

等價關係會把集合分成一塊塊**等價類 (equivalence class)**，合起來剛好是原集合、彼此不重疊，這叫**分割 (partition)**。

例：整數對 3 同餘（a ~ b 若 a、b 除以 3 餘數相同）：

```
餘 0: { …, 0, 3, 6, 9, … }
餘 1: { …, 1, 4, 7, 10, … }     ← 三個等價類，不重疊、合起來是全部
餘 2: { …, 2, 5, 8, 11, … }
```

關鍵：等價類是「分群」，**不是排序**。

## 偏序 → Hasse 圖

偏序描述「不一定每兩個都能比」的順序。畫成 **Hasse 圖**（往上=較大，省略自迴圈與可由遞移推出的邊）。

例：{1,2,3,6} 的「整除」關係：

```
        6
       ╱ ╲
      2   3
       ╲ ╱
        1
```

讀法：1 整除所有；2、3 都整除 6；但 **2 和 3 彼此不可比較**（誰也不整除誰）——這就是「偏序不要求任兩元素可比」。若任兩元素都可比，才是**全序**。

## 函數：合成順序與反函數

函數合成 `g ∘ f` 的意思是「**先做 f，再做 g**」（從右往左讀）：

```
x ──f──▶ f(x) ──g──▶ g(f(x))
(g∘f)(x) = g(f(x))
```

只有**雙射 (bijective)** 才有真正的反函數（單射 + 滿射，才能一對一倒回去）。

## 計數函數個數

從 A 到 B 的函數：每個 A 的元素都要選一個 B 的目標。

```
函數總數      = |B|^|A|                （每個輸入獨立選）
單射函數數    = |B|·(|B|−1)···         （後面的選擇被前面用掉，需 |B|≥|A|）
```

## 常見誤解

- 對稱 (symmetric) 與反對稱 (antisymmetric) **不是互斥**，某些關係可同時滿足（例如只有 (a,a) 的關係）。
- 等價類形成分割，**不是排序**。
- 偏序**不要求**任兩元素可比較（全序才要求）。
- Hasse 圖要**省略**自迴圈和可由遞移推出的邊（別把它們畫上去）。
- 遞移閉包 (transitive closure) 是「包含原關係的**最小**遞移關係」，不是隨便補邊。

## 解題時的判斷

- 關係題：逐一檢查四性質 → 判斷是等價關係還是偏序。
- 等價關係 → 想它切出哪些等價類（分割）。
- 偏序 → 畫 Hasse 圖，找最大/最小、可比/不可比。
- 函數題：先確認 domain/codomain/單射/滿射，合成記得「右先左後」。
