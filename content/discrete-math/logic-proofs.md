# 邏輯與證明

## 這章在學什麼？

如何把「我覺得這是對的」變成「我能嚴謹證明它是對的」。
重點是**挑對證明方法**，並把直覺拆成一步步可檢查的推論。

## 謂詞邏輯：量詞順序會改變意思

`∀`（對所有）、`∃`（存在）。順序一換，意思天差地遠：

```
∀x ∃y (y > x)   「每個 x 都找得到某個更大的 y」  → 真（y 可隨 x 變）
∃y ∀x (y > x)   「存在一個 y 比所有 x 都大」      → 假（沒有最大的數）
```

所以 `∀x∃y` 與 `∃y∀x` **通常不同**，讀題要看清順序。

## 四種主要證明法

**① 直接證明**：從前提一路推到結論。

**② 逆否 (contrapositive)**：要證 `P→Q`，改證等價的 `¬Q→¬P`。當「假設結論不成立」比較好下手時用。

**③ 反證 (contradiction)**：假設「結論為假」，推出矛盾。
例（√2 是無理數，概念）：假設 √2 = a/b（最簡分數）→ 推導出 a、b 都是偶數 → 與「最簡分數」矛盾 → 假設錯 → √2 無理。

**④ 數學歸納**：證「對所有 n 成立」。

## 歸納法走一遍：1+2+…+n = n(n+1)/2

```
① Base case：n=1
   左 = 1，右 = 1·2/2 = 1  ✓

② 歸納假設：假設 n=k 時成立
   1+2+…+k = k(k+1)/2

③ 歸納步驟：證 n=k+1 也成立
   1+2+…+k+(k+1)
   = k(k+1)/2 + (k+1)      ← 代入假設
   = (k+1)(k/2 + 1)
   = (k+1)(k+2)/2          ✓ 正是 n=k+1 的公式

→ 由歸納法，對所有 n≥1 成立。
```

**強歸納 (strong induction)**：證 n 時可用「所有 < n 的情況」，不只用 n−1。

## 不變量 (invariant)：證「達不到」很好用

對演算法/狀態轉換題：找一個「初始成立、每步操作後仍成立」的性質。若目標狀態違反它，就證明了**目標不可達**。

## 推翻全稱命題：一個反例就夠

要否定「所有 X 都 P」，找**一個** X 不 P 即可，不必證很多。

## 常見誤解

- 逆命題 `Q→P` ≠ 原命題（別拿來當證明）。
- 反證法是假設「結論**為假**」，不是假設它成立。
- 歸納法**不能只驗幾個例子**就說成立——一定要有歸納步驟。
- 量詞順序 `∀∃` vs `∃∀` 意思不同，別互換。

## 解題時的判斷

- 先寫出 P、Q，量詞的 domain 是什麼。
- 選證法：能直接推就直接；「結論否定較好用」→ 逆否/反證；「對所有 n」→ 歸納。
- 用反例時，確認它**真的滿足前提、又違反結論**。
