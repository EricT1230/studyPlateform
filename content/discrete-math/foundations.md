# 離散數學基礎

## 這章在學什麼？

離散數學是「把日常推理寫成精確符號」的語言：邏輯、集合、函數、關係。
考試常不是考你算多快，而是考你能不能**把文字翻成正確的符號**。

## 命題邏輯：蘊含最容易錯

`P → Q`（如果 P 就 Q）只有一種情況為假：**P 真但 Q 假**。

```
 P  Q │ P→Q
 T  T │  T
 T  F │  F     ← 唯一為假
 F  T │  T     （P 假時，整句一律為真）
 F  F │  T
```

關鍵改寫：`P → Q` 等價於 `¬P ∨ Q`。
易錯：`P → Q` 和它的逆命題 `Q → P` **不等價**；但和**逆否** `¬Q → ¬P` 等價。

## 量詞否定：否定要「換量詞 + 否定述詞」

```
¬(∀x P(x))  ⇔  ∃x ¬P(x)      「並非全部都 P」= 「存在一個不 P」
¬(∃x P(x))  ⇔  ∀x ¬P(x)      「不存在 P」    = 「全部都不 P」
```

易錯：否定時不能只改最後的述詞、量詞不動——量詞要一起翻轉。

## 集合運算

```
聯集 A∪B：在 A 或在 B
交集 A∩B：同時在 A 和 B
差集 A−B：在 A 但不在 B
補集 A^c：不在 A
```

De Morgan（和邏輯的對應）：
```
(A∪B)^c = A^c ∩ B^c
(A∩B)^c = A^c ∪ B^c
```

## 函數：domain / codomain / image

- **單射 (injective)**：不同輸入給不同輸出（不撞）。
- **滿射 (surjective)**：codomain 每個值都有人對應到。
- **雙射 (bijective)**：又單又滿。

有限集合的關鍵限制（鴿籠原理的味道）：
```
|domain| < |codomain|  →  不可能滿射（但可能單射）
|domain| > |codomain|  →  不可能單射（必有兩個撞同一個）
```

## 關係：四個性質要逐一檢查

| 性質 | 意思 |
|---|---|
| 反身 reflexive | 每個 a 都有 a~a |
| 對稱 symmetric | a~b 則 b~a |
| 反對稱 antisymmetric | a~b 且 b~a 則 a=b |
| 遞移 transitive | a~b 且 b~c 則 a~c |

- **等價關係** = 反身 + 對稱 + 遞移（把集合切成等價類）。
- **偏序** = 反身 + 反對稱 + 遞移（不要求任兩元素可比較）。
- **全序**：偏序 + 「任兩元素都可比較」。

## 證明方法選擇

- **直接證明**：從前提一路推到結論。
- **反證 (contradiction)**：假設結論為假，推出矛盾。
- **逆否 (contrapositive)**：證 `¬Q → ¬P`（和原命題等價，有時更好證）。
- **數學歸納**：證 base case + 「假設 n 成立 → n+1 成立」。
- **反例**：要推翻全稱命題，舉一個反例就夠。

## 常見誤解

- 逆命題 ≠ 原命題（`P→Q` 不等於 `Q→P`）。
- 量詞否定要連量詞一起翻。
- 偏序**不要求**任兩元素可比較（全序才要求）。

## 解題時的判斷

- 先把每個符號的精確意思寫出來。
- 要證的是全稱、存在還是條件命題？（決定證法）
- 集合/函數是否有限？（決定能不能用鴿籠/大小限制）
- 用公式或定理前，先確認前提滿足。
